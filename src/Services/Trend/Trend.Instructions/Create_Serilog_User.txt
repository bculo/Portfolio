7.
Watch carefully and map properties to appsettings.json

  appsettings.json
  SerilogMongo:AuthDatabase = adming (property db from MongoDB)
  SerilogMongo:AuthMechanisam = SCRAM-SHA-1 (property credentials from MongoDB)
  SerilogMongo:Database = TrendLogs (property roles from MongoDB)
  SerilogMongo:UserName = logger (property user from MongoDB)	
  SerilogMongo:Password = loggerpassword (given when creating logger user)
